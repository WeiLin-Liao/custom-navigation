## 如何使用？

### 一、将项目克隆到本地

``` bash
git clone https://github.com/WeiLin-Liao/custom-navigation.git
```

### 二、安装依赖
``` bash
cd custom-navigation
npm install
```

### 三、编译

``` bash
npm run dev || wepy build --watch
```
编译后生成dist目录 使用微信开发者工具导入项目 文件夹指定dist目录即可
